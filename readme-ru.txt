КРАТКОЕ ОПИСАНИЕ

AMVONET Room - это модуль для СДО Moodle, который предназначен для совместной работы Moodle (тесты, оценки, форумы, авторизация и др.) и Виртуальных классных комнат.
После установки и регистрации плагина AMVONET ROOM пользователи получают возможность:
- участвовать во стречах, семинарах, вебинарах посредством Интернет, используя функционал Виртуальной классной комнаты. Взаимодействие пользователей в Виртуальной классной комнате происходит при помощи веб-камеры, микрофона, чата, скриншаринга, собраузинга и публикации контента.
- записывать сессии и просматривать и редактировать их в последствии.

========================================
УСТАНОВКА

1. Распакуйте архив в папку 'mod', которая находится в корневой директории Moodle (например, /usr/share/moodle/mod).
2. Авторизуйтесь в Moodle с правами администратора и откройте страницу /admin (например, http://myschool.com/admin) для обновления базы данных Moodle данными, необходимыми для работы плагина.
3. С правами администратора откройте настройки плагина в меню Modules -> Activites -> AMVONET Room и кликните по ссылке 'Register'.
4. На открывшейся странице введите вашу регистрационную информацию и получите лицензионный ключ.
5. Вернитесь в настройки плагина (Modules -> Activites -> AMVONET Room), введите лицензионный ключ в поле 'Key' и сохраните изменения.
6. Если все действия проделаны корректно, то в настройках плагина отобразится статус 'Active'. Успешной Вам работы с AMVONET!

========================================
ИСПОЛЬЗОВАНИЕ AMVONET Room

1. Авторизуйтесь в Moodle в качестве учителя.
2. Зайдите в курс, в котором вы хотите создать новую Виртуальную классную комнату.
3. Нажмите кнопку 'Turn editing on' и выберите 'AMVONET Room' в выпадающем списке 'Add activity'.
4. Внесите всю необходимую информацию для новой Виртуальной классной комнаты AMVONET Room: название, краткое описание, модератор и др.
5. Нажмите кнопку 'Save and display'.
6. Для входа в сессию нажмите 'Enter to session'.
