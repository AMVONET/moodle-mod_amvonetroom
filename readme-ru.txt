КРАТКОЕ ОПИСАНИЕ

КОМНАТА AMVONET - это модуль, который позволяет получить доступ к функциям системы AMVONET, таким как:
- Общая авторизация в AMVONET и в Moodle под одной учетной записью.
- Интеграция с функциями "Календарь" и "Оценки" системы Moodle
- Веб-конференции при помощи аудио, видео и приватных аудио-чатов.
- Модуль презентаций с возможностью создания рабочих досок и загрузки файлов PowerPoint, PDF, видео и других типов.
- Скриншаринг с возможностью аннотировать происходящее.
- соБраузинг, т.е. возможность синхронно показывать какой-либо веб-сайт всем участникам сессии.
- Публичный и приватный текстовые чаты.
- Возможность разбивать на обособленные группы участников сессии.
- Возможность получать обратную связь от студентов в реальном времени, а также проводить опросы и оценивать студентов.
- Возможность публиковать файлы, которая интегрирована с файлами, размещаемыми непосредственно в Moodle.
- Возможность записывать сессии и просматривать эти записи
- Возможность редактировать записанные сессии.
- Возможность добавлять интерактивные точки, которые делают записанные сессии интерактивными в отличие от простого видео-урока.
- Возможность создавать из записанных сессий пакеты в формате SCORM
- И многое другое!

Для получения дополнительной информации пожалуйста посетите наш сайт http://www.amvonet.com

========================================
СИСТЕМНЫЕ ТРЕБОВАНИЯ

Для сервера:
- Moodle 1.9.*
- Разрешены исходящие сетевые соединения на файрволе на портах: 1935, 8080 и 80
- PHP 5

Для клиентской части:
- Операционная система Windows XP или выше ИЛИ MacOS X 10.5 или выше
- Adobe Flash Player 10.0 или выше
- Любой Интернет браузер, поддерживающий Adobe Flash Player или приложение Amvonet Enterprise, установленное на компьютере
- Активированные куки в браузере, если будет использоваться браузерный клиент

УСТАНОВКА

КОММЕНТАРИЙ: Вы должны иметь административный доступ к серверу, на котором установлен Moodle. Для информации смотрите:  http://docs.moodle.org/en/Installing_contributed_modules_or_plugins

1. Скачайте архив и распакуйте его в папку 'mod', которая находится в корневой директории Moodle (например, /usr/share/moodle/mod).
2. Авторизуйтесь в Moodle с правами администратора и откройте страницу /admin (например, http://myschool.com/admin) для обновления базы данных Moodle данными, необходимыми для работы плагина.
3. С правами администратора откройте в панели 'Администрирование' настройки плагина в меню Модули -> Элементы курса -> AMVONET Комната 
4. Откроется страница под названием Комната AMVONET. Под надписью 'Статус: Не зарегистрирован' кликните по ссылке 'Зарегистрировать модуль'.
5. Откроется страница "AMVONET - Register Plugin". На странице введите Вашу регистрационную информацию, короткое описание Вашей школы и укажите домен, на котором располагается Ваша школа с Moodle. После этого нажмите Register plugin
6. В ответ вы получите письмо с подтверждением регистрации. Пройдите по ссылке, указанной в письме для завершения регистрации учетной записи.
7. Откроется новая страница "Your schools and plugins" с информацией о вашей школе. Нажмите на ссылку "Activate" рядом с полем "Product key". Вы будете перенапралены в Moodle на страницу Модули -> Элементы курса -> AMVONET Комната. Лицензионный ключ будет автоматически подставлен в свойства плагина.
8. Если все действия проделаны корректно, то в настройках плагина отобразится статус 'Активный'. Успешной Вам работы с AMVONET!

ИСПОЛЬЗОВАНИЕ КОМНАТЫ AMVONET

1. Авторизуйтесь в Moodle в качестве учителя.
2. Зайдите в курс, в котором вы хотите создать новую Виртуальную классную комнату.
3. Нажмите кнопку 'Редактировать' и выберите 'AMVONET Комната' в выпадающем списке 'Добавить элемент курса...'.
4. Внесите всю необходимую информацию для новой Виртуальной классной комнаты AMVONET: название, краткое описание, модератор и др.
5. Нажмите кнопку 'Сохранить и показать'.
6. Для входа в сессию нажмите 'Войти в комнату AMVONET'.

Для получения дополнительной информации пожалуйста обратитесь к руководствам пользователя, перечисленным ниже

РАЗЛИЧИЯ МЕЖДУ БРАУЗЕРНЫМ КЛИЕНТОМ И ПРИЛОЖЕНИЕМ AMVONET ENTERPRISE

Вы можете работать с AMVONET используя веб-браузер или посредством приложения AMVONET Enterprise. 
Браузерный клиент может быть запущен в любом веб-браузере, который поддерживает работу с Adobe Flash Player (например, Internet Explorer, Firefox, Crome).
AMVONET Enterprise - это настольное приложение, которое устанавливается на компьютер отдельно и работает под операционными системами Windows или Mac OS X.

AMVONET Enterprise рекомендуется, если в дополнение к функциям браузерного клиента Вы хотите использовать следующие функции:
- Скриншаринг
- соБраузинг
- Переключение Аудио-видео режимов: двунаправленное аудио-видео, аудио конференция, общий микрофон
- Загрузка в презентацию расширенного набора типов файлов: офисные документы (doc, xls, ppt), аудио (mp3), видео (avi, mp4, mov, mpeg, mkv) и других.

AMVONET Enterprise можно скачать со страницы входа в сессию нажатием на ссылку 'Скачать AMVONET Enterprise' в правом верхнем углу страницы
Вы также можете скачать приложение AMVONET Enterprise по следующим ссылкам:
- Для Windows: http://download.amvonet.com/getamvonet/?version=4.5&platform=win
- Для Mac OS X: http://download.amvonet.com/getamvonet/?version=4.5&platform=mac
