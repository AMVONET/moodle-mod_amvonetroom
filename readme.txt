AMVONET ROOM plugin for Moodle

AMVONET ROOM Copyright (C) 2004-2011 AVE Intervision

========================================
SHORT DESCRIPTION

The AMVONET plugin integrates AMVONET Connect and AMVONET Publish with Moodle. The plugin allows access to AMVONET features such as:
- Single Sign On Integration with Moodle.
- Web conferencing with audio and video capabilities as well as private audio chat capabilities.
- Presentation module for uploading powerpoints, pdfs, whiteboards and more.
- Ability to screenshare with annotation tools.
- Share website browsing in real time with the cobrowsing module.
- Public and private text chat.
- Easily control permissions of users for whiteboard, public and private text chat, audio and video.
- Ability to create break out groups from inside of a session.
- Realtime student response system and integration with moodle quizzes.
- File sharing including integration with Moodle course files.
- Ability to record and edit a session.
- Ability to add ipoints that can make a recorded session interactive.
- And more!

========================================
SYSTEM REQUIREMENTS

For server:
- Moodle 1.9.*
- Allowed outgoing network connections in Firewall to ports: 1935, 8080 and 80
- PHP 5

For client:
- MacOS or Windows operating system
- Adobe Flash Player 10.0 or greater
- Internet browser which supports Adobe Flash Player (Internet Explorer, Firefox or Crome are recommended) or Amvonet Enterprise Rich client installed
- Cookies enabled for Internet browser client

INSTALLATION GUIDE
NOTE: You must have access to the server containing your Moodle. For more details see:  http://docs.moodle.org/en/Installing_contributed_modules_or_plugins

1. Download the zip file and unzip it to your moodle/mod directory (for example /usr/share/moodle/mod).
2. Log into your Moodle site as the administrator and go to /admin page to update Moodle's database (for example http://myschool.com/admin).
3. Under the Site Administration block click on Modules -> Activites -> AMVONET Room.
4. A new page labeled AMVONET Room will appear. Under the text "Status: Not Registered" click on the link "Register".
5. On the new page enter your information and click the Register Button.
6. You will receive an email with confirmation of your registration, click on the link in the email to activate your registration.
7. At the top right of this page click on the Register Plugin hyperlink.
8. Enter a title for your moodle site and domain where your Moodle is installed, and click the Register Button at the bottom.
9. Copy the key that appears in the key field and then on your moodle page go back to modules -> Activites -> AMVONET Room, put the license key into 'Key' field and save changes.
10. Click on Register, if everything is good to go then the unregistered status will be changed to active and you can begin to use the plugin! We hope you will enjoy using AMVONET!

========================================
USING AMVONET ROOM ACTIVITY

1. Log into Moodle as a teacher.
2. Go to course where you want to create new activity.
3. Click 'Turn editing on' and select 'AMVONET Room' in the 'Add an activity' dropdown.
4. Put in all the necessary data for 'AMVONET Room': live session name, description, moderator and other.
5. Click the 'Save and display' button.
6. Click the 'Enter to session' button.

DIFFERENCES BETWEEN AMVONET BROWSER CLIENT AND AMVONET ENTERPRISE APPLICATION

You can work with AMVONET by using a web browser or the AMVONET Enterprise Application. 
The web browser client may be used by any web browser that supports the Adobe Flash Player (for example Internet Explorer, Firefox, Crome).
The AMOVNET Application is a desktop application which works under Windows or Mac OS X.
The AMVONET Application is recomended if you wish to use such features as:
- Sreensharing
- coBrowsing
- Audio-video modes: two way audio-video, audio conference, open mic
- Extended list of supported file types which might be uploaded to the presentation (office docs, mp3, video files).

The AMVONET Application can be downloaded from the enter to session page by clicking on the 'Download AMVONET Enterprise' link at the top right of the page or you may download it through the following links:
- For Windows: http://download.amvonet.com/getamvonet/?version=4.5&platform=win
- For Mac OS X: http://download.amvonet.com/getamvonet/?version=4.5&platform=mac
